# Termination and Concession

Use this card when a debate risks becoming theater.

## Loop Stop

- Default to one constructive round, one crossfire, and one rebuttal cycle.
- Continue only if new evidence appears or a participant changes a claim.
- If positions are unchanged after two exchanges, stop and preserve dissent.

## Falsifiable Dissent

Every live objection must include one of:

- a file, command, source, test, or observation that would prove it
- a missing check that would decide it
- an explicit `speculation` label

Drop objections that cannot name what would change the answer.

## Concession

Concession is a good outcome, not a loss. When evidence lands, update the decision record:

- what changed
- which evidence changed it
- which risk remains
- whether the next step is action, more research, or preserved dissent
