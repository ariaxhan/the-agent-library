# Public Forum

Use for binary decisions where both sides are plausible.

Classic shape:
- coin flip controls either side or speaking order
- pro/aff constructive
- con/neg constructive
- crossfire
- rebuttal
- summary
- grand crossfire
- final focus

Agent adaptation:
1. Write `Resolved: <decision>`.
2. Flip/randomize roles when possible.
3. Pro must build the best case for action.
4. Con must build the best case against action.
5. Crossfire asks short pointed questions.
6. Final focus cannot introduce new arguments.
7. Judge by weighing impacts, reversibility, evidence, and risk.

Use when the danger is everyone agreeing too fast.
