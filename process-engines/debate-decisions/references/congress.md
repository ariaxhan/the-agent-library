# Congressional Debate

Use for governance, prioritization, doctrine, roadmap decisions, and commission shaping.

Classic shape:
- chamber considers legislation
- presiding officer manages speaking order
- speeches alternate aff/neg where possible
- questioning follows speeches
- precedence/recency prevents one speaker from dominating
- chamber votes, amends, tables, or refers

Agent adaptation:
1. Draft a bill or resolution.
2. Assign a presiding officer.
3. Let speakers give short aff/neg speeches.
4. Require each speech to add a new argument, amendment, or weighing lens.
5. Allow amendments when the bill is directionally right but too broad.
6. Vote with recorded reasons.

Output:
```text
Bill:
Amendments:
Aff case:
Neg case:
Questions:
Vote:
Implementation order:
Minority report:
```

Use when a decision affects shared process or future agents.
