# Platform Speech

## Informative

Use to teach without selling too hard.

Shape:
- hook
- map
- 2-4 clear teaching blocks
- example per block
- takeaway

Good for explaining systems, research, architecture, and tools.

## Original Oratory / Original Advocacy

Use when the goal is persuasion plus action.

Shape:
- arresting hook
- problem or contradiction
- why the audience currently misses it
- proof
- solution or call
- memorable close

Original oratory is broader persuasion. Original advocacy is more explicitly "do this."

Agent adaptation:
- make the claim brave but supportable
- include evidence without sounding like a report
- avoid five-point sprawl
- end with an action the audience could actually take

Use for doctrine, manifestos, Medium essays, pitch scripts, public-facing positions, and "why this matters" artifacts.

## Commentary

Use for concise perspective on a situation.

Shape:
- trend or event
- hidden pattern
- consequence
- recommendation
