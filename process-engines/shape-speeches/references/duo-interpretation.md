# Duo and Interpretation

## Duo

Use when two voices create meaning together.

Classic duo interpretation uses two performers with no props, no touching, and no eye contact. The important transfer is not the stage rule; it is disciplined interdependence. Each voice must need the other.

Agent adaptation:
- Voice A can be thesis, builder, witness, or system voice.
- Voice B can be skeptic, user, future self, antagonist, or conscience.
- Alternate intentionally; do not just split paragraphs.
- Include one clash point and one convergence point.

Good for:
- two-agent presentations
- Claude/Codex joint handoffs
- pro/con synthesis
- public explanation with a skeptical interlocutor
- "what I thought vs what reality showed" pieces

## Program Oral Interpretation

Use when multiple sources need to become one argument.

Pattern:
- source 1 establishes the world
- source 2 complicates it
- source 3 gives the turn
- original narration ties them together

Agent adaptation:
- collage quotes sparingly
- paraphrase more than quote
- keep transitions explicit
- make the program argue, not just collect

## Prose, Poetry, Storytelling

Use for human resonance, memory, and mythic/narrative clarity.

Do not overuse. Use when plain explanation is true but not sticky.
