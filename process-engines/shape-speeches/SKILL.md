# Speech Protocols

Use competitive speech formats as shaping tools for expression. Debate protocols decide between options; speech protocols make an idea land.

Do not wait for direct invocation. When the task asks for a brief, pitch, article, talk track, meeting script, handoff, synthesis, advocacy memo, or public-facing explanation, choose the smallest useful speech pattern.

## Format Picker

- **Impromptu**: use for fast sensemaking from a messy prompt.
- **Extemporaneous Speaking**: use for sourced briefs and current-context recommendations.
- **Informative Speaking**: use to teach a system, concept, or research result clearly.
- **Original Oratory / Original Advocacy**: use to persuade, motivate action, or crystallize doctrine.
- **Duo**: use when two voices should co-present, contrast, interrupt, or braid perspectives.
- **Program Oral Interpretation**: use to collage multiple sources into one argument.
- **Prose / Poetry / Storytelling**: use for narrative, reflective, or emotionally precise artifacts.
- **Commentary**: use for concise analysis of a trend, incident, or situation.

Detailed format cards live in `references/`. Load only the relevant card.

## Core Rules

1. Decide the audience before drafting.
2. Decide the speech job: explain, persuade, move, brief, synthesize, or perform.
3. Build a spine: hook, thesis, points, impact, close.
4. Keep the spoken shape even for written output: short beats, clean transitions, memorable close.
5. For advocacy, name the action requested.
6. For duo, make the second voice necessary, not decorative.
7. For impromptu, prefer a sharp frame over exhaustive coverage.
